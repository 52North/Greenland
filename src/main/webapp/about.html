Greenland is a dynamic map viewer for uncertainty- and quality-aware geodata. For more information, visit our Wiki

This development is supported by the European Commissions's Seventh Framework Programme (FP7) via the projects GeoViQua (QUAlity aware VIsualisation for the Global Earth Observation system of systems), grant agreement number 265178, and UncertWeb, grant agreement number 248488.




This application uses icons of the GeoSilk icon set by Rolando Peate, based on the Silk icon set.

"Greenland" is part of the 52°North Geostatistics Community
